Tips
====
- All filenames are lowercase, with hyphens between words.
- Use .jpeg (not .jpg). They are the same format but different extensions.
- Recommended size: roughly square, at least 300x300 pixels.
- Bigger photos make the page load slower; resize to ~600px max
  for fastest loading.
- After adding photos: just refresh the page. No code change needed.
